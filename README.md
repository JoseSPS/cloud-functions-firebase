# cloud-functions-firebase
Project to know cloud functions on Firebase

# Resources
* (Material Icon)[https://google.github.io/material-design-icons/]

## Before strat need to install
* Node.js
* CLI firebase
*  _npm install -g firebase-tools_

## Deploy firebase
* _firebase login_
* _firebase init_
* _firebase serve_
* _firebase deploy_ 

## Deploy Only Functions
* _firebase deploy --only functions_
